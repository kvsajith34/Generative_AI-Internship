# Runway Workflow Guide

This guide explains how to use the generated `outputs/runway_broll_prompts.md` file with Runway ML to create final B-roll clips for your reels.

---

## What You Need

- A Runway ML account with available credits
- The `outputs/runway_broll_prompts.md` file generated by ViralClip AI
- A video editor (CapCut, Premiere Pro, DaVinci Resolve, etc.)
- The corresponding `outputs/subtitles/reel_N.srt` files

---

## Step-by-Step Runway Workflow

### 1. Open Runway

Go to [https://runwayml.com](https://runwayml.com) and log in.

### 2. Navigate to Text to Video

Select **Text to Video** or **Gen-3** from the dashboard.

### 3. Copy a Prompt

Open `outputs/runway_broll_prompts.md`. Each reel has a polished prompt ready to paste. For example:

```
Cinematic 5-second B-roll clip. Close-up of a frustrated creator staring at a complex editing timeline on a large monitor. Dim room, blue glow from the screen. Smooth camera motion, shallow depth of field, professional color grading, soft ambient lighting, 4K, highly detailed, editorial style.
```

### 4. Set Duration and Aspect Ratio

- **Duration:** 5 seconds
- **Aspect ratio:** 9:16 for YouTube Shorts / Instagram Reels / TikTok

### 5. Generate the Clip

Click generate. Review the result.

### 6. Iterate If Needed

If the clip does not match your vision, adjust the prompt slightly. Change one descriptor at a time to save credits.

### 7. Download and Repeat

Download the clip and repeat for reels 2 through 5.

---

## Credit-Saving Tips

1. **Test one prompt first.** Do not generate all five clips until you are happy with the visual style.
2. **Keep visual style consistent.** Use the same lighting and color-grade words across all prompts.
3. **Use 5-second clips.** Longer clips cost more credits and are usually unnecessary for B-roll.
4. **Save your best prompts.** Reuse them for future videos in the same niche.
5. **Use image references when possible.** If you have a reference image, use Runway's Image to Video for more control.

---

## Syncing B-roll With Your Reel

1. Import the talking-head footage into your editor.
2. Place the Runway B-roll clip on a track above.
3. Import the matching `reel_N.srt` subtitle file.
4. Add the suggested on-screen text at the right moment.
5. Match the B-roll mood to the spoken content (e.g., calm scene for reflective moments, fast cuts for energetic hooks).

## Important Note

ViralClip AI does **not** call the Runway API directly. It prepares professional prompts that you paste into Runway. This keeps the app lightweight, free to run in demo mode, and flexible across different Runway versions.
